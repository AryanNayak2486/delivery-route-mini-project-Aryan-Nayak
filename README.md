# Logistics Unlocked: The Delivery Route Capstone

## The Project
In this capstone, I built an end-to-end delivery optimization engine modeled after platforms like Amazon. It integrates five different algorithmic strategies into a single logistics pipeline.

## The Optimization Pipeline
1. **Selection**: Using Greedy logic to prioritize high-value parcels based on vehicle weight capacity.
2. **Scheduling**: Applying Dynamic Programming to ensure all deliveries fall within their specific time windows.
3. **Pathfinding**: Leveraging Dijkstra’s algorithm to find the shortest path to each customer.
4. **The Final Mile**: Solving the TSP to determine the most efficient sequence for a driver to return to the warehouse.

## Visualizations
The project includes generated route maps and profit-vs-weight charts, providing a clear picture of how algorithmic choices impact the bottom line in a delivery business.
